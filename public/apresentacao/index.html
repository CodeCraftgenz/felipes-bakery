✨ Diferenciais

O que torna este projeto especial

🎠

Carrossel de Combos Sazonais

Cards com scroll horizontal suave, setas de navegação e snap CSS. 1 card mobile, 2 tablet, 3 desktop — zero JavaScript de terceiros.

📸

Upload de Imagens do Dispositivo

Admin faz upload da foto do pão direto do celular. Arquivo salvo em volume Docker persistente, servido em rota dedicada.

🏷️

Vitrine Pública de Cupons

Cupons ativos aparecem automaticamente na home para os clientes copiarem com um clique — sem nenhuma ação extra do admin.

🔒

Autenticação Dupla

Admin usa e-mail/senha com sessão de 8h. Clientes acessam via e-mail/senha ou Google OAuth com sessão de 7 dias.

📊

Analytics em Tempo Real

Gráficos Recharts com receita diária e mensal. Top clientes por valor. Estoque crítico com alerta visual. Tudo sem ferramentas externas pagas.

🐳

Deploy Docker Completo

Imagem multi-stage com Next.js standalone. Docker Compose com MySQL, volumes persistentes e dumb-init como PID 1.
